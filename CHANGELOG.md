# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-15

### Added

#### Authentication
- User registration with email/password
- Secure login with JWT tokens
- HTTP-only cookie authentication
- Password strength validation
- Session management

#### Dashboard
- Real-time metrics dashboard
- Activity feed with team updates
- Project overview cards
- Team velocity sparklines
- Quick action buttons

#### Project Management
- Create and manage multiple projects
- Project color coding
- Task progress tracking
- Member management

#### Kanban Board
- Four-column board (Backlog, In Progress, Review, Completed)
- Drag-and-drop task management
- Task priority badges
- Assignee avatars
- Due date tracking
- Inline task creation

#### Team Directory
- Team member profiles
- Task completion statistics
- Member invitation system
- Role-based access

#### Analytics
- Team velocity charts
- Task distribution pie charts
- Sprint burndown graphs
- Performance metrics
- Weekly trends

#### Settings
- Profile management
- Notification preferences
- Theme selection (dark/light/system)
- Security settings

#### UI Components
- Button (4 variants, 3 sizes, loading states)
- Input (with icons, password toggle, search)
- Card (interactive, elevated, ghost variants)
- Modal (multiple sizes, backdrop blur)
- Avatar (gradient initials, status indicators)
- Badge (multiple variants with dot option)
- Dropdown (portal-rendered, animated)
- Select (custom styled)
- Toast (4 types with progress bar)
- Tooltip (smart positioning)

### Technical
- React 18 with Vite
- Express.js backend
- SQLite database with better-sqlite3
- Zustand state management
- CSS Modules styling
- Recharts visualizations
- Lucide icons
- JWT authentication
- bcrypt password hashing
